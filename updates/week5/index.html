Week 5 Team Updates

Sensing team:

This class we completed the same experiment as last class but used the camera to classify each color in the tube using HSL and hex values. We performed the nitrate tests on three different concentration groups 25, 50, and 100 ppm to get a total of 12 different values. We also added the nitrate Solution 2 from the expired API test kit to a clear test tube to see how the solids in the solution would settle if left undisturbed.

Found that needed to be greater than 1m in order to accurately measure and not give minimum values of around 1m. But success overall! No problem with the measuring cone. Additionally, the code we used is uploaded in the code>esp32 pathway of the github.
